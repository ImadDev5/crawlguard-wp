AI content monetization and bot detection for WordPress.

== Description ==

CrawlGuard WP is an advanced WordPress plugin that detects AI bots and monetizes their traffic while protecting your content.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/crawlguard-fixed` directory
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the CrawlGuard screen to configure the plugin
